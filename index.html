Toronto Migration Timeline

This is still a work in progress, I just wanted to publish this first part while it could still be useful for the current spring migration.

Below is a list of "mass arrival" dates of each species to Toronto. The method to calculate these dates is based on Mike Burrell's blog post. Instead of using all available eBird data, only the past 20 years were used to have a more accurate prediction based on current circumstances.

More regions will be calculated in the near future, as well as some graphs for each species to show how their migration dates have changed from the past.
