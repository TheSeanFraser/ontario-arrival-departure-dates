Migration Timelines

This project is intended to predict when the highest number of each species arrives in each region of Ontario. The method to calculate these dates is based on Mike Burrell's blog post. Instead of using eBird data from every available year, only the past 20 years were used to have a more accurate prediction based on current circumstances.

In the future, fall migration will be included - as well as some graphs for each species to show how their migration dates have changed from the past.

All bird data is from eBird.

eBird. 2021. eBird: An online database of bird distribution and abundance [web application]. eBird, Cornell Lab of Ornithology, Ithaca, New York. Available: http://www.ebird.org. (Accessed: April 5, 2022).
